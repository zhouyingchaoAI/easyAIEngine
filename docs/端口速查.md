# 端口速查

| 服务 | 默认端口 | 说明 | 常用入口 |
| --- | --- | --- | --- |
| Web 管理界面 | 7901 | 统一控制台与配置界面 | `http://localhost:7901` |
| 实时检测服务 | 7902 | 人数统计 / 客流分析 / 人头检测接口 | `http://localhost:7902/infer` |
| 绊线统计服务 | 7903 | 绊线人数分析接口 | `http://localhost:7903/infer` |

## 快速启动命令

- `./scripts/start_manager.sh` – 启动 Web 管理界面  
- `./scripts/start_algorithm_service.sh` – 启动实时检测服务  
- `./scripts/start_line_crossing_service.sh` – 启动绊线统计服务
